This project presents an interactive Mental Health Analysis Dashboard developed using Power BI. The goal is to analyze mental health indicators such as anxiety and depression across different years, age groups, and geographical regions.

The dashboard transforms raw data into meaningful insights using data visualization techniques, enabling better understanding of trends, patterns, and distributions in mental health conditions.

🎯 Objectives
Analyze year-wise trends of mental health indicators
Study distribution of anxiety vs depression
Compare impact across age groups (category analysis)
Perform comparative analysis across multiple dimensions
Visualize geographical distribution using Shape Map
📊 Dashboard Features
🔹 Overview Page
KPI Cards:
Total Records
Average Mental Health %
Maximum & Minimum Values
Trend Analysis (Line Chart)
Distribution Analysis (Donut Chart)
Filters (State, Mental Health Type, Year)
🔹 Advanced Insights Page
Category Analysis (Bar Chart – Age Groups)
Comparative Analysis (Line + Bar Charts)
Geographical Analysis (Shape Map)
Key Insights Section
📁 Dataset Information
Source: CDC (Centers for Disease Control and Prevention) via Data.gov
Format: CSV
Initial Records: 16,765
After Cleaning: 11,430
🔑 Key Attributes:
Year / Time Period
Mental Health Indicator (Anxiety / Depression)
Age Group / Category
State (Geographical Data)
Percentage Values
⚙️ Data Preprocessing Steps
Handling missing values
Removing duplicates
Data type conversion
Standardizing column formats
Creating calculated measures (Average %, etc.)
📈 Key Insights
Anxiety shows consistently higher values than depression
Peak mental health impact observed around 2022–2023
Younger age groups (18–29 years) are most affected
Mental health conditions vary significantly across states
Distribution is uneven, indicating need for targeted interventions
🛠️ Tools & Technologies
Power BI (Data Visualization)
